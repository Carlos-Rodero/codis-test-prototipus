## SD
La carpeta SD conté el codi per obrir i tancar un arxiu dins de la SD. Necessita introduïr una tarjeta SD en format FAT16 o FAT32. Controla els següents Pins:
- P4: SD_CMD
- P8: SD_DAT
- P23: SD_CLK
